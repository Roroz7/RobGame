# RobGame - by Robin Pit

Un site web de mini-jeux moderne et responsive, optimisé pour mobile et PC.

## 🎮 Jeux Disponibles

- **Snake Classic** - Le classique jeu du serpent avec contrôles tactiles
- **Tetris Pro** - Puzzle de blocs avec système de niveaux
- **Flappy Bird** - Jeu d'adresse avec physique réaliste
- **2048 Challenge** - Puzzle mathématique addictif
- **Memory Master** - Jeu de mémoire avec cartes
- **Pong Retro** - Tennis de table contre IA

## 🚀 Fonctionnalités

- ✅ Interface moderne et professionnelle
- ✅ Design responsive (mobile-first)
- ✅ Contrôles tactiles optimisés
- ✅ Animations fluides
- ✅ Système de score intégré
- ✅ Compatible GitHub Pages
- ✅ Performance optimisée

## 🛠️ Technologies

- **HTML5 Canvas** pour les jeux
- **CSS3** avec variables CSS et animations
- **JavaScript ES6+** avec classes modernes
- **Font Awesome** pour les icônes
- **Google Fonts** (Poppins)

## 📱 Contrôles

### PC
- **Flèches directionnelles** ou **WASD** pour tous les jeux
- **ESPACE** pour actions spéciales (saut, rotation, etc.)
- **ESC** pour fermer les jeux

### Mobile
- **Glissement tactile** dans les 4 directions
- **Tap** pour actions spéciales
- Interface optimisée pour écrans tactiles

## 🎯 Installation

1. Clonez ou téléchargez le projet
2. Ouvrez `index.html` dans un navigateur
3. Ou déployez sur GitHub Pages pour un accès en ligne

## 🌐 Déploiement GitHub Pages

1. Créez un repository GitHub
2. Uploadez tous les fichiers
3. Activez GitHub Pages dans les paramètres
4. Votre site sera accessible à `https://username.github.io/repository-name`

## 🎨 Personnalisation

Le design utilise des variables CSS dans `:root` pour faciliter la personnalisation des couleurs et thèmes.
